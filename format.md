# Format amb Markdown (H1)

Això és un petit exemple de com escriure en **Markdown (strong)**. Entre parèntesi s'indiquen els equivalents en HTML (el primer cop que apareixen).

## Exemples (H2)

Aquest text és en *cursiva (em)* i aquest en **negreta**.  
També podem combinar-los: ***negreta i cursiva***.  
I fins i tot ~~ratllar text (s)~~ si cal.

---
(hr)

### Llista no ordenada (ul)

- Element **1** (li)
- Element *2*
- Element ~~3~~ (eliminat)
- Element amb **parts** *diferents* de format

### Llista numerada (ol)

1. Primer pas (li)
2. Segon pas
3. Tercer pas (ul imbricat)
   - Subpunt amb guió (li)
   - Un altre subpunt

---

> *Consell:* El Markdown és molt útil per escriure documentació clara i fàcil de llegir. (div amb estil)
